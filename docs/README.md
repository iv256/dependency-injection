# Documentation

This directory contains the main architectural,
conceptual, and project evolution documentation
for the *Dependency-Injection* container.

---

| Document | Description |

| [`glossary.md`](./glossary.md) | Definitions of core terminology, runtime concepts, dependency model vocabulary, and execution semantics terms. |
| [`examples-index.md`](./examples-index.md) | Description of the shared fictional application world used across examples, tests, and documentation. |

| [`../examples/README.md`](../examples/README.md) | Executable examples overview and runnable dependency resolution scenarios. |
